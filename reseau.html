Configuration Réseau et Accès à Distance

1. Passage en mode "Accès par Pont"

Nous avons modifié la configuration réseau de la VM pour utiliser le mode Bridged Adapter.

Justification technique : En mode NAT, la VM est isolée. Le mode Pont connecte la carte réseau virtuelle directement au réseau physique local . Cela permet à la VM d'obtenir sa propre adresse IP (distincte de l'hôte), la rendant accessible comme un serveur indépendant.

Adresse IP obtenue : 192.168.1.16

2. Configuration SSH et Transfert SCP

Nous avons installé OpenSSH (sudo apt install openssh-server) et configuré le fichier /etc/ssh/sshd_config pour autoriser la connexion .

Transfert des fichiers avec SCP

Pour transférer notre dossier "Site Web" du Mac vers la VM, nous avons utilisé la commande scp :

scp -O -r "/Users/wzngs/Documents/projetweb" admin_Gr102_Binome06@192.168.1.16:/tmp
  • -r : Transfert récursif (dossier entier).
  • path local : Chemin du dossier sur le Mac.
  • user@ip : Destination sur la VM.

Serveur Web Apache

Nous avons installé le serveur web avec sudo apt install apache2.

Ensuite, nous avons configuré les droits pour que notre groupe puisse modifier le site :

sudo chown -R www-data:Binome06 /var/www/html
sudo chmod -R 775 /var/www/html
sudo systemctl reload apache2
